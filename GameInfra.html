IT Architecture & Tools

GoTechUp's IT architecture leverages cutting-edge technologies to ensure a smooth, scalable, and efficient game development and deployment process. Below is an overview of the core systems used within GoTechUp, including their functions and installation steps.

1. Unity 6

Overview

Unity 6 is the game engine used for developing GoTechUp’s interactive simulations. It is a powerful, cross-platform engine that integrates various game elements, such as graphics, physics, and interactions, to provide an immersive experience.

Function

- Handles the rendering, real-time physics, and multiplayer interactions.
- Used for building, testing, and deploying GoTechUp’s simulation games.

Installation Steps

1. Download Unity Hub from the Unity website.
2. Sign in or create a Unity account.
3. Go to the 'Installs' tab in Unity Hub and click the 'Add' button.
4. Select Unity 6 from the available versions.
5. Install additional modules based on your platform (e.g., Windows, Mac, WebGL).

2. Plastic SCM

Overview

Plastic SCM is a version control system (VCS) that helps manage and track changes in the game’s source code and assets. It supports branching, merging, and team collaboration for smoother version management.

Function

- Manages game source code and assets.
- Enables multiple developers to work simultaneously without overwriting each other’s work.
- Allows rollback to previous game versions when necessary.

Installation Steps

1. Download the installer from the Plastic SCM website.
2. Run the installer and follow the on-screen instructions.
3. Sign in to Plastic SCM and set up or connect to a repository.

3. Combu

Overview

Combu is an advanced tool for compressing and optimizing game assets. It ensures media files are compressed efficiently, optimizing both performance and storage.

Function

- Compresses textures, models, and game assets.
- Reduces overall file size while maintaining high quality.
- Improves game load times and reduces storage requirements.

Installation Steps

1. Download Combu from the official website.
2. Follow the installation wizard to install the software.
3. Launch Combu and configure the settings based on GoTechUp’s project needs.

4. AWS Server

Overview

Amazon Web Services (AWS) provides the cloud infrastructure for GoTechUp’s multiplayer and backend services. AWS supports hosting servers, managing databases, and scaling game infrastructure as needed.

Function

- Scales servers to meet demand for seamless multiplayer experiences.
- Provides cloud storage, compute power, and data analytics.
- Manages user data and game functions efficiently.

Setup Steps

1. Create an AWS account on the AWS Registration Page.
2. Access the AWS Management Console after setting up your account.
3. Select necessary AWS services (e.g., EC2 for servers, S3 for storage, RDS for databases).
4. Follow the instructions to configure and deploy your game services.

5. Adminer PHP

Overview

Adminer PHP is a lightweight database management tool with a web interface for managing SQL databases, including MySQL, PostgreSQL, and SQLite.

Function

- Used to perform operations such as data retrieval, insertion, and schema modifications.
- Provides an easy-to-use interface for database management.

Installation Steps

1. Download Adminer from the official website.
2. Upload the PHP file to your server (no installation required).
3. Access Adminer via a web browser (e.g., `http://yourdomain/adminer.php`).
4. Log in with your database credentials to manage your databases.
